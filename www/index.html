relay.bitcraftsync.app

A community SpacetimeDB relay/proxy for BitCraft. It holds a single persistent subscription to each upstream game region and fans it out to any number of downstream clients from a sibling SpacetimeDB — so community tools (BitCraftMap, BitCraftSync, BitJita, …) share one upstream cost instead of each holding their own subscription. Point your SpacetimeDB SDK at the addresses below; it’s a drop-in wss:// replacement.

SpacetimeDB relays

Each row is a live mirrored region. Connect your SpacetimeDB SDK with the SDK address as the host and the Database as the database name, or open the full subscribe URL for raw-WebSocket testing. The frontend negotiates whichever subprotocol the client offers — v1.bsatn.spacetimedb (synthesises full upstream-flavored TransactionUpdates) or v2.bsatn.spacetimedb (plain passthrough). See the README for protocol details. Ports with no relay behind them return nginx 502; only sources reported live by /health are listed.

Convenience HTTP endpoints

Read-only JSON endpoints for quick lookups across all mirrored regions. Every result row carries an __source field naming the region it came from. Lists are capped at 100 rows; narrow with the query params.

Endpoint Returns Query params
/health Fleet source map: per-source port, mirror database name, schema_cached flag, and total schema_count.
$ curl https://relay.bitcraftsync.app/health
